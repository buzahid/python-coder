About This App

Model Coder is a Python sandbox that runs entirely in your browser using PyScript. Experiment with Python code using numpy, pandas, matplotlib, and scikit-learn — no installation required.

To get started, select a sample from the dropdown or write your own code in the editor pane, then click the run button. Happy coding!
